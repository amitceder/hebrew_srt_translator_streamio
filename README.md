# Hebrew SRT Translator + Stremio addon

This service translates English or Arabic SRT subtitles into Hebrew with the
existing Google baseline plus Groq contextual-polish pipeline. It also exposes
a Stremio subtitle addon that resolves a source subtitle through OpenSubtitles
and returns a cached Hebrew SRT.

See [STREMIO.md](STREMIO.md) for HTPC Docker setup, stable HTTPS hosting, and
Stremio installation/publication instructions.
